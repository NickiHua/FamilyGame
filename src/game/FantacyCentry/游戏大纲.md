# 幻世录类战棋游戏 —— 《幻世纪》

## 一、游戏概述

- **类型**: 回合制战棋RPG（SRPG）
- **视角**: 正交俯视角，32×32网格
- **平台**: Windows
- **技术**: Python + Pygame
- **核心玩法**: 回合制战棋战斗 + 大地图探索 + 剧情驱动
- **关卡数**: 35关（主线25关 + 支线/自由战斗10关）
- **章节数**: 5章

---

## 二、世界观

大陆名为「辉耀大陆」，千年前封印了异界邪神「虚无之主」。如今帝国「铁焰王朝」暴政横行，背后是复活邪神的黑暗教团「幻影之环」在操纵。封印正在崩塌，各地出现魔物异变。

主角陆离是边境猎村少年，身怀上古封印者的血脉而不自知，在家园被毁后踏上旅途，逐渐聚集伙伴，从一个普通少年成长为拯救大陆的英雄。

## 五、大地图系统

仿照幻世录的大地图系统：

- 每章解锁新的大地图区域
- 大地图上有**主线关卡节点**（必须通关推进剧情）和**支线节点**（可选）
- 有**自由战斗点**（练级场），可反复挑战，敌人等级随玩家提升
- 地图节点间有路径连接，部分支线需要满足条件解锁
- 城镇节点：可购买道具、查看角色、触发支线对话

### 大地图区域

| 章节 | 区域名 | 包含节点数 | 自由战斗点 |
|------|--------|-----------|-----------|
| 序章 | 火烬村周边 | 3关 | 1个 |
| 第一章 | 苍原荒野→风谷 | 7关 | 2个 |
| 第二章 | 铁焰王城周边 | 8关 | 2个 |
| 第三章 | 南境+诸国联盟 | 9关 | 3个 |
| 终章 | 虚无裂隙 | 8关 | 2个 |

## 七、成长与养成系统

### 经验与升级
- 战斗中获得经验值（击杀/攻击/治疗/辅助均可获得）
- 每5级可学习新技能
- 15级触发职业进阶（转职）

### 装备系统
- 武器、防具、饰品三个槽位
- 城镇商店购买 + 关卡宝箱 + 战斗掉落

### 技能系统
- 每角色有4个主动技能 + 2个被动技能
- 技能随等级解锁

---

## 八、自由度设计

1. **大地图自由移动**: 已解锁的区域可自由回访
2. **自由战斗点**: 每个区域2-3个练级点，敌人等级动态调整（玩家等级±2）
3. **支线关卡**: 部分关卡可选，提供额外奖励和剧情
4. **角色好感度**: 特定关卡中的选择影响角色对话（不影响主线）
5. **隐藏要素**: 满足条件解锁隐藏商店/装备
